# todolist api server

Flask project for study

- Flask
- pipenv
- MySQL
- Frontend: https://github.com/daisyleenr/todolist-react

# Requirements

python >= 3.7
pipenv
mysql >= 5.7

# Install modules

```
$ pipenv install
```

# Activate this project's virtualenv

```
$ pipenv shell
```

# Set DB STRING

```
$ export TODO_DB_STRING='mysql://user:password@localhost/todolist?charset=utf8'
```

# Run

```
$ python app.py
```
